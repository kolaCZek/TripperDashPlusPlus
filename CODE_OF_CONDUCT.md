# Code of Conduct

## TL;DR

**Be excellent to each other.** This is a hobby project. Constructive feedback welcome, snark and personal attacks not. Disagree on technical merits all you want; don't make it personal.

## What's expected

- Assume good faith. Most people aren't out to attack you.
- Critique code, ideas, decisions — not people.
- Use inclusive language. Avoid assumptions about gender, nationality, experience level, or bike preference.
- If you spot a mistake (in code, in docs, in someone's tone), call it out kindly.

## What's not OK

- Personal attacks, harassment, discriminatory language.
- Trolling, deliberately disruptive behavior, sustained off-topic derailment.
- Sharing private info without consent.
- Anything that would get you a yellow card in a calm bar.

## Enforcement

If someone violates this:

1. **First offense:** I'll DM them privately and ask them to walk it back.
2. **Repeat or severe:** Comment removed, possible block from the repo.
3. **Egregious (harassment, doxxing, etc.):** Immediate block + GitHub ToS report.

There's no committee; it's just me ([@kolaCZek](https://github.com/kolaCZek)). I'll try to be fair. If you think I've been unfair to you, push back — privately first, then escalate to GitHub support if needed.

## Reporting

Report problems to [@kolaCZek](https://github.com/kolaCZek) via GitHub DM, or by email to the address on my GitHub profile. I'll respond within a few days. Reports are kept confidential.
